# Building the RPMs

Before the MISP RPM can be created, the following dependencies must be built (in this order)
* php
* php-pear
* php-pear-CommandLine
* php-pear-Crypt_GPG
* php-redis
* python-cybox
* python-stix
* python-mixbox
* python-pydeep
* python-pymisp
* python-lief (my current version doesn't work correctly, work in progress)
* misp
